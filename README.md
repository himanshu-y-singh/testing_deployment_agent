# React Login App

This is a simple React login page built using Vite.

## 🚀 Getting Started

### 1. Install dependencies
```
npm install
```

### 2. Run the development server
```
npm run dev
```

### 3. Open in browser
```
http://localhost:5173
```

## 📁 Project Structure

- src/
  - App.jsx
  - main.jsx
  - components/
    - Login.jsx

## ✨ Features

- Simple login form
- Email & password input
- Basic form handling
