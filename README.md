# React Login & Signup App

This is a simple React login and signup page built using Vite.

## 🚀 Getting Started

### 1. Install dependencies
```
npm install
```

### 2. Run the development server
```
npm run dev
```

### 3. Open in browser
```
http://localhost:5173
```

## 📁 Project Structure

- src/
  - App.jsx
  - main.jsx
  - components/
    - Login.jsx
    - Signup.jsx

## ✨ Features

- Login & Signup forms
- Toggle between login/signup
- Basic form handling
